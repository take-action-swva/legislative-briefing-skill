# Briefing QA Checklist

Use after each AI-generated briefing before distributing to group leaders.
**Check every box or note why it was skipped.**

Briefing topic: ____________________________________________   Date: ________________

State: ____________________________________________   Reviewer: ____________________________________________

---

## 1. Output Format

- [ ] Briefing was produced as a **.docx file** (not plain markdown or text)
  - *Download as PDF from Google Docs for fully clickable links on any device*
- [ ] Citations use inline **[source]** hyperlinks throughout
  - *Not Word footnotes — footnotes are not tappable in Google Docs*
- [ ] Document opens and renders correctly in Google Docs

---

## 2. Document Structure

Every required section below should be present. Optional sections should be
present when applicable. A missing required section is a red flag.

**Required:**
- [ ] Summary present
- [ ] Current Status present
- [ ] [State] Senators' Positions present
- [ ] [State] House Members / Key Votes present
- [ ] [State]-Specific Impact present
- [ ] Who to Contact present
- [ ] Recommended Actions — Right Now present and numbered
- [ ] Accuracy Notes present

**Optional (include when applicable):**
- [ ] Committee Leverage present if any state member sits on the relevant committee
- [ ] Watch List present if pivotal swing votes exist outside the state delegation

---

## 3. Accuracy & Sources

**This is the most important section. If any box cannot be checked, the
briefing should not be distributed.**

### Bill / EO Identity

- [ ] Exact bill number confirmed *(e.g., H.R. 22, not just "the SAVE Act")*
- [ ] Congress session noted *(e.g., 119th Congress, 2025–2026)*
- [ ] Confirmed this is the current version *(not a prior or superseded version)*
- [ ] If House and Senate companion bills exist, the correct one is cited for each claim

### Status Verification

- [ ] Current status verified at [congress.gov](https://congress.gov) or
  [senate.gov](https://senate.gov) — not only from a news article
- [ ] If bill has been on Senate floor, procedural vote details confirmed at
  [dailypress.senate.gov](https://www.dailypress.senate.gov) — not only from
  news summaries
- [ ] Status is current *(checked within the past week; flag if older)*
- [ ] Next procedural moment identified *(upcoming vote, hearing, recess, deadline)*

### Member Positions

- [ ] Every stated member position has a source
  *(press release, floor statement, or vote record)*
- [ ] No positions inferred from party affiliation alone
- [ ] Unstated positions are noted as "position not publicly stated" — not left blank
- [ ] District numbers and member assignments verified at the state's political
  data source *(redistricting litigation can change these mid-Congress — see
  references/sources.md for the correct source for this state)*

### Statistics & Impact Claims

- [ ] All statistics have a source link
- [ ] State-specific data used where available *(not only national statistics)*
- [ ] Advocacy org sources *(Brennan Center, Campaign Legal Center, Democracy
  Docket, etc.)* are used for context/analysis, not for stating legislative status
- [ ] Reconciliation pathway claims (if present) flagged in Accuracy Notes
  unless confirmed via a procedural primary source such as a CRS report or
  legislativeprocedure.com analysis

### Accuracy Notes Section

- [ ] Any unverified claims are flagged in Accuracy Notes
- [ ] Date of last status check is stated
- [ ] Briefing flagged as potentially outdated if more than one week old

---

## 4. Actionability

A briefing without a clear call to action is incomplete for this audience.

- [ ] Recommended Actions are numbered and ranked by urgency
- [ ] Each action is specific
  *(names a person to call, a URL to visit, a deadline to meet)*
- [ ] "Who to Contact" section distinguishes allies to thank from targets to pressure
- [ ] At least one action is doable **right now** — not just "watch and wait"

---

## 5. Final Check Before Distributing

- [ ] Ran a final status check at [congress.gov](https://congress.gov) since research began
- [ ] Checked for active court challenges at [democracydocket.com](https://democracydocket.com)
  *(for voting rights / election bills — litigation often moves faster than legislation)*
- [ ] Checked for new court rulings or injunctions on any active EOs
- [ ] PDF exported from Google Docs and links tested on mobile
- [ ] Briefing date is visible and accurate
